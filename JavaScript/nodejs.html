命令行

参数解释器

  • yargs-parser 强大的命令参数解释器
  • minimist 解析参数选项，没有任何奇特的修饰。
  • cac 用于构建命令行应用程序的简单而强大的框架。
  • mri 快速扫描CLI标志和参数，minimist 和 yargs 解析器的快速，轻量级替代方案。
  • has-flag 检查argv是否具有特定标志
  • dargs 反转 minimist。 将选项对象转换为命令行参数数组。
  • sade 用于构建快速、响应迅速且有用的命令行界面 CLI 应用程序！
  • argparse CLI 参数解析器
  • meow CLI 帮助参数解析器

命令行程序解决方案

  • yargs 解释命令行参数。
  • commander 命令行程序的完整解决方案。
  • meow 命令行帮助。
  • sade 生成漂亮的命令行帮助。
